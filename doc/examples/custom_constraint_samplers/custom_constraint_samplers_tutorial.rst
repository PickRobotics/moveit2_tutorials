Custom Constraint Samplers
==========================

.. note:: 이 튜터리얼은 작업중입니다. 개선하는데 도움을 줄 수 있다면 부탁합니다!

개요
--------
일부 계획 문제는 더 복잡하거나 더 어려운 계획 문제를 위해 커스텀 제약 샘플러를 필요로 합니다.
이 문서는 MoveIt에서 사용하는 커스텀 모션 계획 제약 샘플러를 만드는 방법을 설명합니다.
